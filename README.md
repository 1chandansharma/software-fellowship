# 🚀 9-Day Software Fellowship Journey

Welcome to my journey through an intensive, hands-on 9-day Software Fellowship program focused on fullstack development, modern software engineering practices, and real-world project implementation.

## 🧠 Program Overview

This bootcamp is designed to provide industry-relevant skills through daily labs, practical tasks, and collaborative coding. From building UIs and APIs to deploying apps and integrating AI, this repo documents it all.

- **APIs**: OpenWeather, Gemini, OpenAI
